[Docker] Docker 설치 및 시작하기

2021/05/01 카테고리 Infra, DevOps, Docker

설치

Mac OS 환경에서는 Homebrew로 Docker를 쉽게 설치할 수 있다.

도커 버전을 확인해보자.

도커 이미지 다운로드

우선 다운받을 수 있는 도커 이미지를 검색해보자.

docker pull명령어로 도커 이미지를 다운받을 수 있다. centos:7 이미지를 다운받아보자.

docker images 명령어로 다운받은 도커 이미지 목록을 확인할 수 있다.

도커 컨테이너 생성

docker create 명령어로 도커 컨테이너를 생성할 수 있다. 자세한 실행 옵션은 이 곳에서 확인할 수 있다.

도커 컨케이너 상태 및 목록 확인

docker ps명령어로 도커 컨테이너 목록을 확인할 수 있다. 아무 옵션을 추가하지 않으면 실행 중인 컨테이너 목록만 출력된다. 정지된 컨테이너 목록까지 확인하려면 -a 옵션을 추가한다.

STATUS는 컨테이너의 상태를 나타내며 다음 값들을 가질 수 있다.

  • Up: 실행 중
  • Exited: 종료된 상태
  • Pause: 일시 중지된 상태

도커 컨테이너 실행

docker start 명령어로 도커 컨테이너를 시작한다.

docker attach명령어로 컨테이너 내부로 접속할 수 있다.

exit을 입력하여 컨테이너에서 빠져나올 수 있다.

주의할 점은 exit을 사용하여 컨테이너에서 빠져나오면 컨테이너가 종료된다는 것이다.

컨테이너를 종료시키지 않고 빠져나오려면 Ctrl + P, Q를 입력한다.

docker run 명령어

docker run 명령어를 사용하면 도커 이미지 다운, 도커 컨테이너 생성, 시작, 접속을 한 번에 할 수 있다.

도커 컨테이너 정지

docker stop 명령어를 사용하면 도커 컨테이너를 정지할 수 있다.

도커 컨테이너 삭제

docker rm 명령어를 사용하면 도커 컨테이너를 삭제할 수 있다.

docker container prune 명령어를 사용하면 모든 도커 컨테이너를 삭제한다.
