[Kotlin] Kotlin Compiler

2018/07/01 카테고리 Kotlin

Java Compiler와 JVM

Java Compiler는 .java확장자가 붙은 소스코드를 .class확장자가 붙은 바이트코드로 변환한다. 이 바이트코드는 JVM에서 실행할 수 있다.

Kotlin Compiler와 JVM

Kotlin Compiler는 .kt확장자가 붙은 소스코드를 .class확장자가 붙은 바이트코드로 변환한다. 이 바이트코드 역시 JVM에서 실행할 수 있다.
