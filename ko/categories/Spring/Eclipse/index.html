[Eclipse] Eclipse에서 다양한 Spring 프로젝트 생성하기

2019/02/01 카테고리 Spring, Eclipse

STS

Eclipse 환경에서는 STS(Spring Tool Suite)을 설치하여 Spring 프로젝트를 쉽게 구성할 수 있습니다. Eclipse > Help > Eclipse Marketplace로 이동합니다. 그리고 검색창에 STS를 입력하고 적절한 버전을 설치합니다. 이 포스트에서는 STS 3을 설치합니다.

Spring 프로젝트 생성

Eclipse > File > New > Project로 이동하여 spring을 검색합니다.

  • Spring Legacy Project: Spring Boot를 사용하지 않는 Spring 프로젝트를 구성할 때 사용합니다.
  • Spring Starter Project: Spring Boot 프로젝트를 구성할 때 사용합니다.

1. Spring Legacy Project

Spring Legacy Project를 선택하고 Next >버튼을 누르면 다음 화면으로 이동합니다.

2. Spring Starter Project

Spring Starter Project를 선택하고 Next >버튼을 누르면 다음 화면으로 이동합니다.
