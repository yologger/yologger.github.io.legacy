[Java] 객체 비교하기, Comparable, Comparator

2018/03/20 카테고리 Java

Primitive Type 비교하기

Java에서 Primitive Type를 비교할 때는 ==을 사용한다.

Reference Type 비교하기

Reference Type을 비교할 때 ==를 사용하면 객체의 주소값을 비교한다.

객체가 아닌 값을 비교할 때는 Object클래스의 equals()메소드를 사용한다.

다만 객체에서 equals()를 직접 오버라이드 해야한다.

Comparable

배열을 정렬할 때는 기준이 있어야한다. 예를 들어 Person클래스가 있다고 가정하자.

이제 Arrays.sort()를 사용하여 배열을 정렬하려고 한다.

Comparable인터페이스와 compareTo()를 구현하여 정렬의 기준을 정의할 수 있다.

이제 Person클래스의 age속성으로 정렬하도록 Comparable을 구현해보자.

이제 Comparrable.compareTo()을 사용하여 정렬된다.

Comparator

Comparator를 사용하여 정렬의 기준을 정의할 수도 있다.
