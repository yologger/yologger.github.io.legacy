확장 함수

확장 함수(Extension Function)는 기존에 정의된 클래스에 함수를 추가하는 것입니다.

보통 Java API나 다른 사람이 만든 라이브러리처럼 코드를 직접 변경하기 어려운 경우 확장 함수을 사용합니다. 다음은 Int클래스에 홀수인지 짝수인지 판단해주는 함수를 추가하는 예제입니다.
