[Kotlin] 형 변환(Type Casting)

2018/09/06 카테고리 Kotlin

형 변환

형 변환은 자료형을 다른 자료형으로 변환하는 것을 의미합니다.

(1) 기본자료형의 형변환

기본 자료형을 형변환할 때는 내장된 형변환 함수를 사용합니다.

(2) 클래스 인스턴스의 형변환

클래스 인스턴스를 형변환할 때는 키워드 as를 사용합니다.

부모클래스가 Nullable타입이면 키워드 as?를 사용합니다.
