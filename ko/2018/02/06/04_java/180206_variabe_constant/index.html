[Java] 변수와 상수

2018/02/06 카테고리 Java

변수

변수는 자료형을 변수명 앞에 붙여서 선언한다.

변수는 값을 초기화한 후 변경할 수 있다.

상수

상수는 자료형 앞에 키워드 final를 붙인다. 보통 상수의 이름은 대문자와 언더바(_)를 사용한다.

상수는 값을 한번 할당하면 변경할 수 없다.
