[Java] 원시 타입(Primitive type)과 참조 타입(Reference type)

2018/02/02 카테고리 Java

Java에는 크게 두 종류의 자료형이 있다.

  • 원시 타입(Primitive Type)
  • 참조 타입(Reference Type)

원시 타입

Java가 제공하는 원시 타입(Primitive type)은 다음과 같다.

  • byte
  • short
  • int
  • log
  • float
  • double
  • boolean
  • char
