[Java] 키워드 final

2018/02/17 카테고리 Java

final

키워드final은 크게 세 가지 용도로 사용된다.

(1) 상수

키워드 final은 변수를 값을 변경할 수 없는 상수로 만드는데 사용한다.

이렇게 선언된 상수는 값을 변경할 수 없다.

(2) 상속 불가능한 클래스

부모 클래스에 final을 붙여보자.

이 클래스는 상속할 수 없다.

(3) 오버라이드 불가능한 메소드

부모 클래스 메소드 앞에 final을 붙으면 오버라이딩이 불가능하다.
