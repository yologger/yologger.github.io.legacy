[Android Jetpack] Data Binding 개념 및 사용법

2020/07/05 카테고리 Android, Jetpack, Architecture Component

Jetpack

Jetpack은 더 좋은 품질의 안드로이드 개발을 돕는 라이브러리의 집합입니다.

Jetpack은 크게 네 가지 카테고리로 분류됩니다.

  • Architecture Component
  • UI Component
  • Foundation Component
  • Behavior Component

이번 포스트에서는 Lifecycle Owner와 Lifecycle-aware Component에 대해 알아보겠습니다.

등장 배경

다음과 같은 레이아웃 XML 파일이 있다고 가정합시다.

보통 액티비티에서는 findViewById()를 통해 뷰를 객채화합니다. 그리고 객체화한 뷰의 변수나 메소드로 UI를 수정합니다.

Data Binding을 사용하면 UI 요소와 데이터를 보다 쉽게 결합할 수 있습니다.
