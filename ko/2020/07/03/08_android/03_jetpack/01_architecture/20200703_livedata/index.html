2020/07/03 카테고리 Android, Jetpack, Architecture Component

Jetpack

Jetpack은 더 좋은 품질의 안드로이드 개발을 돕는 라이브러리의 집합입니다.

Jetpack은 크게 네 가지 카테고리로 분류됩니다.

  • Architecture Component
  • UI Component
  • Foundation Component
  • Behavior Component

이번 포스트에서는 Lifecycle Owner와 Lifecycle-aware Component에 대해 알아보겠습니다.

LiveData

LiveData는 관찰 가능한 데이터입니다. LiveData는 value라는 속성에 데이터를 유지하고 있으며, 이 데이터의 값이 변경되면 관찰자에게 통지됩니다.
