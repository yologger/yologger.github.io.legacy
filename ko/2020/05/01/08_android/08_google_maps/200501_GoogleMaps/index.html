[구글 지도(Google Maps)] (1) 구글 지도 띄우기

2020/05/01 카테고리 Android, Google Maps

들어가며

일곱 개의 포스트에 걸쳐 구글 지도(Google Maps)에 대해 알아보고 있습니다.

  • 구글 지도 띄우기
  • 지도 타입 변경하기
  • 마커 보여주기
  • 지도 확대와 축소 (줌-인과 줌-아웃)
  • 지도에 내 위치 보여주기
  • 내 위치로 카메라 이동하기
  • 맵 컨트롤

1. SHA-1 인증키 발급

Google Cloud Platform, Google Play, Google Firebase 등의 서비스를 사용하다보면 SHA-1 인증키가 필요한 경우가 많습니다. SHA-1 인증키는 안드로이드 스튜디오에서 아래와 같이 발급할 수 있습니다.

우선 프로젝트를 열고 [Gradle] > [Task] > [android] > [signingReport]를 클릭합니다.

그럼 하단 로그에 다음과 같이 SHA1 인증키가 출력됩니다.

2. API Key 발급

Google Maps API를 사용하려면 API Key를 발급받아야 합니다. https://cloud.google.com/에서 로그인하고 오른쪽 상단에 있는 콘솔 버튼을 눌러줍니다.

API 및 서비스에 들어갑니다.

프로젝트를 하나도 추가하지 않았다면 다음과 같은 화면이 나옵니다. 오른쪽 프로젝트 만들기를 클릭합니다.

프로젝트 이름을 입력하고 만들기를 클릭합니다.

프로젝트가 생성되었습니다. 왼쪽 사이드 바 메뉴에서 [API 및 서비스] > [대시보드]로 들어갑니다

상단 중앙의 API 및 서비스 사용 설정을 클릭합니다.

Maps SDK for Android를 선택합니다.

사용 버튼을 누릅니다.

왼쪽 메뉴에서 사용자 인증 정보를 누르고 상단의 사용자 인증 정보 만들기를 클릭한 후 API 키를 선택합니다.

키 제한을 클릭합니다. 키 제한은 실제 배포되는 앱에서 키가 무단으로 사용되는 것을 방지하기 위함입니다.

키의 이름은 임의로 설정합니다.

키를 Android 앱에서만 사용할 수 있도록 제한할 수 있습니다.

Google Cloud Platform은 여러 API를 제공합니다. 이 중에서 안드로이드를 위한 맵 API만 사용하도록 제한할 수도 있습니다.

아래와 같이 앱의 사용량도 제한할 수 있습니다. 이때 항목 추가를 누르고 위 안드로이드 스튜디오 프로젝트에서 발급한 SHA-1 인증키와 패키지 이름을 입력합니다. 이후 완료 버튼을 누르고 하단의 저장 버튼을 누릅니다.

3. 안드로이드 스튜디오에서 구글 맵스 설정

Google Maps Android API를 사용하려면 Google Play Services 라이브러리를 설치해야합니다.

안드로이드 스튜디오에서 SDK Manager에 들어가서 SDK Tools를 선택합니다. Google player services를 체크하고 OK 버튼을 눌러 설치합니다.

이제 모듈 수준의 build.gradle에 의존성을 추가해줍니다.
