git checkout

현재 작업 중인 Commit에는 HEAD포인터가 표시된다.

다음 결과물은 master브랜치의 가장 최근 Commit의 Id가 44444이며, 현재 이 Commit에서 작업 중이라는 의미다.

이제 git checkout명령어와 Commit Id를 이용하여 특정 Commit으로 이동해보자.

git branch

브랜치(Branch)를 사용하면 작업 공간을 분기할 수 있다.

우선 초기 상태를 확인해보자.

git branch명령어로 현재 작업 중인 브랜치를 확인할 수 있다.

새로운 브랜치를 생성해보자

두 개의 브랜치(develop, master)가 존재하며, 현재 master에서 작업하고 있다.

git checkout명령어를 이용하면 브랜치를 이동할 수 있다. develop로 이동해보자.

HEAD가 develop로 이동했다.

develop브랜치에서 Commit을 해보자.

로그를 확인해보자.

다시 master로 이동해서 Commit을 해보자.

이제 로그를 확인해보자. git log명령어에 --branches와 --graph옵션을 추가하면 시각적으로 브랜치를 확인할 수 있다.
