git checkout

git checkout명령어를 사용하면 Head를 특정 Commit으로 옮길 수 있다.

현재 master Branch의 Commit 내역은 다음과 같다.

Branch의 가장 최근 Commit에는 Branch 이름이 표시된다. 다음 결과는 Commit ID가 44444인 Commit이 가장 최근 Commit이라는 의미다.

현재 작업 중인 Commit에는 HEAD포인터가 표시된다.

다음 결과물은 master Branch의 가장 최근 Commit의 ID가 44444이며, 현재 이 Commit에서 작업 중이라는 의미다.

이제 git checkout명령어와 Commit ID를 이용하여 특정 Commit으로 이동해보자.

git branch

Branch를 사용하면 작업 공간을 분기할 수 있다.

우선 작업 공간의 초기 상태를 확인해보자.

git branch명령어로 현재 작업 중인 Branch를 확인할 수 있다.

새로운 Branch를 생성해보자

두 개의 Branch(develop, master)가 존재하며, 현재 master Branch에서 작업하고 있다.

HEAD가 develop Branch로 이동했다.

develop Branch에서 Commit을 해보자.
