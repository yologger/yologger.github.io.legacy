[Servlet, JSP] JSP란?

2019/01/11 카테고리 Spring, Servlet, JSP

JSP

JSP(Java Server Pages)는 HTML에 Java 코드를 삽입하여 동적으로 웹 페이지를 생성하는 언어입니다. 아래는 JSP의 예제입니다.

JSP의 동작 원리

JSP는 다음과 같은 과정으로 처리됩니다.

  1. Web container는 JSP 파일을 Servlet으로 변환합니다.
    • MyPage.jsp > MyPage_jsp.java
  2. Servlet 내부에서 JSP 파일의 내용을 출력합니다. MyPage_jsp.java의 코드 일부를 살펴봅시다.
  3. Web Container는 Servlet을 처리합니다.

요약하자면 JSP는 MyPage.js > MyPage_jsp.java > MyPage_jsp.class > HTML 순서대로 처리됩니다.

JSP tag

JSP tag는 JSP 문서 안에 Java 코드를 삽입할 때 사용합니다.

1. 자바 코드

<% %>를 사용하여 자바 코드를 추가할 수 있습니다.

2. 주석

주석은 <%-- --> 형태로 사용합니다.

3. 메소드, 변수 선언

<%! %>는 변수를 선언하거나 함수를 정의할 때 사용합니다.

4. 표현식

JSP에서 무언가를 출력하려면 out.println()을 사용해야합니다.
<%! %>는 변수를 선언하거나 함수를 정의할 때 사용합니다.

<%= %>를 사용하면 더 쉽게 출력할 수 있습니다.

5. 지시자

지시자는 JSP 페이지의 속성을 지정할 때 사용합니다. <%@ %> 형태로 사용합니다.

(1-1) page

page 속성은 해당 페이지의 속성을 지정할 때 사용합니다.

(1-2) include

페이지에 다른 페이지를 삽입할 때 사용합니다.

(1-3) taglib

태그 라이브러리를 사용할 때 사용합니다.

JSP Action Tag

JSP Action Tag는 JSP 페이지 내에서 특정 동작을 하도록 지시하는 태그입니다.

(1) include 태그

현재 페이지에 다른 페이지를 삽입합니다.

(2) forward 태그

현재 페이지에서 다른 페이지로 전환합니다.

(3) param 태그

forward 태그나 include 태그를 사용할 때 데이터를 전달하기 위해 사용합니다.

(4) Bean 객체

JSP 페이지에서 사용할 Java 객체를 Bean이라고 합니다. 다음 Person객체를 JSP 페이지에 만들어봅시다.

<jsp:useBean> 태그를 사용하면 JSP 페이지에서 사용할 객체를 생성할 수 있습니다.
