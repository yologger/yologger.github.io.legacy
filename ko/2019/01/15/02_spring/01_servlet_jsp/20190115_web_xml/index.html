web.xml

WAS는 Java Web Application가 시작하면 web.xml파일을 참고하여 Web Container와 Context를 구성합니다. Eclipse로 생성한 Java Web Application 프로젝트에서는 보통 다음 경로에 위치합니다.

web.xml에는 다음 정보가 포함됩니다.

1. Welcome File

Welcome File은 Web Application의 기본 웹 페이지입니다. 클라이언트가 서블릿 이름을 생략하여 요청하면 웹 서버는 Welcome File을 찾아서 응답합니다.
